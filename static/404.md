### 关于本组件

```mirai-public-api```使用```go```编写，提供支持高并发的公共分发API，用于```mirai```项目的公共API服务(面向使用者的插件中心)。

#### 插件列表获取

```http request
https://miraiapi.jasonczc.cn/getPluginList
```

#### 插件详细信息获取

```http request
https://miraiapi.jasonczc.cn/getPluginDetailedInfo?name=mirai-native
```
